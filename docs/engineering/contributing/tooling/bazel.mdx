Bazel is no longer the primary build or test workflow for Unkey. The
repository keeps some Bazel files temporarily to keep migration pull requests
reviewable, but active development uses mise, Rask, Go build, Docker, and
GoReleaser.

## Replacement workflows

Use these pages for active tooling:

- [Mise](/contributing/tooling/mise) covers tool installation and task
  execution.
- [Builds](/contributing/tooling/builds) covers Docker images and release
  image packaging.
- [Testing](/contributing/quality/testing) covers Rask-based Go test runs.

## Common commands

Run the full suite:

```bash
mise run test
```

Run tests for a specific package:

```bash
mise exec -- rask ./pkg/cache
```

### Building

Build all Go artifacts:

```bash
mise run build
```
